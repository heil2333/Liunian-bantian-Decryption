# Liunian-bantian-Decryption
Decrypt the secret language file of uid8114565
# 用于解包uid8114565游戏的脚本。

# 首次部署
所有脚本文件需要在同一文件夹下。
需要安装了Python（用于解密），pip需要安装依赖pyperclip。
'''
pip install pyperclip
'''
7z确保能在终端打开。

首次使用需要设置文件2和3。

## 2的文件需要设置文件目录
:: 设置变量解压缩目录
set "CACHE_DIR=F:\Zipcache"
:: 设置变量归类已完成的文件
set "DONE_DIR=F:\down\#LinkCache"

## 3的文件需要设置文件目录
'''
:: 配置参数解压游戏目录
set "target_dir=F:\down\FirstPlay"
:: 配置参数解压缓存文件路径
set "cache_dir=F:\Zipcache"

# 如何食用
## 兽语
去up主的评论区复制兽语，双击运行【1提取密文至txt剪贴板.bat】
例如剪贴板内容：者音兽兽兽兽音音音音兽音兽音音译音译音译兽译音者音者译者兽者音兽音兽者译
控制台返回的结果
'''
========================================
      兽语解密工具 - 循环运行
========================================
从剪贴板读取的内容长度: 36
前5个字符: 者音兽兽兽
后5个字符: 兽音兽者译
提取的字典: ['兽', '音', '译', '者']
期望的前缀: 者音兽, 后缀: 译
实际的前缀: 者音兽, 后缀: 译

解密结果:
1234

按Enter键退出...
'''

## 链接文件的提取
将下载好的文件拖拽到【2提取连接exe至txt剪贴板.bat】
你会获得游戏本体的下载连接。

## 密码和二次解压
将下载好的游戏文件拖拽到【3拖拽解压缩HYBHL.bat】
游戏本体文件将会解压到设置的目录下：
'''
:: 配置参数解压游戏目录
set "target_dir=F:\down\FirstPlay"
'''
